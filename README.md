## About

```
Project using nodejs and fastify for an orders api.
The user will be able to apply an CRUD for orders
```

### Node Version
```
v22.11.0
```

### how to run the project
1. Migrate
```shell
npm run migrate:latest
```
2. Start Server
```shell
npm run dev
```

### Technologies
- Nodejs
- Fastify
- Typescript
- Knex
- SQLite
